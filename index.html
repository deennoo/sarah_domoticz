Présentation

Ce plugin est pour le moment l'ébauche d'une passerelle permettant à notre amie Sarah de communiquer avec un serveur Domoticz. Vous pourrez donc récuperer la liste des périphériques, savoir leurs états et les contrôler. Pour le moment n'ayant que très peu de matériels, je ne me suis attaqué qu'au périphérique switch/light ainsi qu'au sensor de type "General", "Temp" et à la météo.

N'hésitez pas à me contacté si cela vous interesse de bosser un peu dessus !

Ce plugin a été conçut en s'inspirant de pas mal d'autre plugin dont un plugin Domoticz réalisé par Michel Rere, je remercie tout le monde pour vos sources d'inspiration et je remercie énormement Jean-Philippe Encausse, qui nous permet à toutes et a tous de retourner en enfance!

Configuration

Configuration du serveur Domoticz

Comme pour la plupart des plugin il vous faudra passer par la case configuration ! Il n'y a rien de bien sorcier, il vous suffit de cliquer sur du porlet present dans le menu général de Sarah.

Une fois dessus, il ne vous reste qu'à saisir l'adresse ip ou se trouve votre serveur domoticz et son port.

Configuration de vos périphériques

Vous allez me dire "c'est bien beau mais ça fait rien du tout ton truc!" Et vous avez raison il vous faut maintenant configurer votre liste de périphérique. Pour cela dites: 

Sarah récupére mes périphériques !

Si tout se passe bien elle devrait modifier le fichier xml pour integrer toute la liste de vos périphériques et ainsi créer les ordres.

Quelques conseils

Pour créer les ordres qui permettront de récupérer des valeurs de vos sensors ou d'allumé ou éteindre vos périphériques, Sarah va récupérer les noms que vous avez saisie lors de la configuration des périphériques sur Domoticz. Donc pensez à nommé vos périphériques de façon a ce que sarah reussisse à bien l'entendre.

Exemple pour: 

Sarah allume la lampe du couloir

Vous devrez nommer le périphérique "la lampe du couloir" dans Domoticz.

Utilisation

Periphériques switch/light

Il n'y a pas plus simple ! Le plugin vous permet 3 choses:

  • Récupérer l'état dans lequel se trouve votre périphérique actuellement
  • Allumer votre périphérique
  • ou l'éteindre

Récuperation de l'état

SARAH quel est l'état de "nom du périphérique"

Allumer

SARAH allume "nom du périphérique"

Eteindre

SARAH éteins "nom du périphérique"

Periphériques sensors

Le plugin vous permet de récupéré la valeur actuel d'un de vos sensor avec la commande 

SARAH donne moi "nom du périphérique"

Météo

Vous pouvez récupérer certaines valeur de météo si vous avez configurer le périphérique "Weather Underground" dans domoticz. Vous aurez donc accès aux commandes: 

SARAH donne moi la température exterieur
SARAH donne moi l'humidité exterieur
SARAH donne moi la préssion exterieur
SARAH donne moi le vent exterieur

Commandes possible

SARAH éteins "nom du périphérique allume le périphérique
